Class Archetype

Archetype represents entities with the same component composition 原型表示具有相同组件组合的实体

Key benefits:

  • Cache-friendly memory layout 缓存友好的内存布局
  • Fast iteration over entities 快速实体迭代
  • Efficient component access 高效组件访问
